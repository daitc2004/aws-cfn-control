
```
Launch and manage CloudFormation templates from the command line

optional arguments:
  -h, --help      show this help message and exit

optional arguments:
  -a              List more info on all stacks
  -b BUCKET       Bucket to upload template to
  -c              Create a stack
  -d              Delete a stack
  -f PARAM_FILE   cfnctl stack parameter file (includes template)
  -l              List stacks
  -nr             Do not rollback
  -p AWS_PROFILE  AWS Profile
  -r REGION       Region name
  -s STACK_NAME   Stack name
  -t TEMPLATE     CFN Template from local file or URL
  -y              On interactive question, force yes
  -v              Verbose config file

```
